# Minecraft Discord Bot A Discord bot for managing a Minecraft server hosted on FalixNodes. ## Features * `/serverstatus` - Check if the Minecraft server is online. * `/serverstart` - Start the server. * `/serverrestart` - Restart the server (admins only). * `/serverstop` - Stop the server (admins only). ## Commands | Command | Description | | ---------------- | ----------------------------------------- | | `/serverstatus` | Shows the current server status. | | `/serverstart` | Starts the Minecraft server. | | `/serverrestart` | Restarts the server (admins only). | | `/serverstop` | Stops the Minecraft server (admins only). | ## Support If you find a bug or have a suggestion, open an issue on this repository. ## License This project is licensed under the MIT License.
